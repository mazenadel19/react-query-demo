# React-Query Demo

React query demo for learning purposes. 

This demo project goes along with Web Dev Simplified YT Video [React Query Makes Writing React Code 200% Better](https://www.youtube.com/watch?v=lVLz_ASqAio&ab_channel=WebDevSimplified).

## Setup

```
npm install -g json-server
json-server --watch db.json
```

```
npm install
npm run dev
```

## Tools

- Axios
- Json-Server
- React Query
- React Router
- Tailwind
- TypeScript
